나만의 퀴즈 게임
프로젝트 개요
Python 콘솔 기반의 퀴즈 게임 프로젝트입니다.
퀴즈 프로그램을 직접 만들며 Python 문법을 학습하고, 클래스 설계와 리팩토링을 통해 게임 형태의 작은 프로젝트를 완성했습니다.
퀴즈 주제 및 선정 이유

주제: 볼링
선정 이유: 볼링에 대해 잘 알고 있는 분야이기 때문에 정확한 문제와 선택지를 구성할 수 있어 선택했습니다.

실행 방법
bashpython3 main.py
기능 목록

퀴즈 풀기 - 등록된 퀴즈를 순차적으로 출제하고 정답을 확인합니다.
퀴즈 추가 - 새로운 퀴즈(문제, 선택지, 정답)를 등록합니다.
퀴즈 목록 - 현재 등록된 퀴즈의 목록을 확인합니다.
점수 확인 - 최고 점수를 확인합니다.
종료 - 데이터를 저장하고 프로그램을 종료합니다.

파일 구조
codyssey_E1-2/
├── main.py        # 메인 실행 파일 (Quiz, QuizGame 클래스 포함)
├── state.json     # 퀴즈 데이터 및 최고 점수 저장 파일
├── .gitignore     # Git 추적 제외 파일 목록
├── doc            # 프로젝트에서 요구한 실행, 환경화면 스크린샷
└── README.md      # 프로젝트 설명 문서
데이터 파일 설명

경로: 프로젝트 루트의 state.json
역할: 최고 점수와 퀴즈 데이터를 저장하여, 프로그램을 종료하고 다시 실행해도 데이터가 유지됩니다.
구조:

quizzes: 퀴즈의 데이터 정보 (문제, 선택지, 정답 번호)
best_score: 최고 점수 (null이면 아직 미플레이 상태)


예시:

json{
    "quizzes": [
        {
            "question": "볼링 핀은 총 몇 개인가?",
            "choices": ["8개", "10개", "12개", "15개"],
            "answer": 2
        }
    ],
    "best_score": null
}
